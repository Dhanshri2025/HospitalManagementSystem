##### \# **Hospital Management System**



**A** **Java console-based Hospital Management System** using **MySQL**.



**Key Features:**

*  CRUD operations for Patients and Doctors.
*  Appointment booking with availability check.
*  Handles multi-word names and foreign key constraints.
*  Menu-driven console interface for easy interaction



**Technologies:**

* Java
* MySQL
* JDBC
